# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. This site showcases your work, skills, and provides a way for visitors to get in touch with you.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling navigation with mobile-friendly hamburger menu
- **Multiple Sections**:
  - Hero section with introduction
  - About Me section
  - Skills showcase with icon cards
  - Featured Projects portfolio
  - Contact information with links
- **Animations**: Fade-in effects and smooth transitions throughout
- **Customizable**: Easy to update with your own information

## Live Demo

View the site by opening `index.html` in your web browser, or deploy it to GitHub Pages (see deployment instructions below).

## Getting Started

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/ty202500368-yabs/Personal-Site.git
   cd Personal-Site
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js http-server
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

### Customization

1. **Update Personal Information**:
   - Edit `index.html` and replace "Your Name" with your actual name
   - Update the hero subtitle with your role/title
   - Modify the About Me section with your bio
   - Update contact information (email, LinkedIn, GitHub links)

2. **Customize Skills**:
   - Edit the skills section in `index.html` to match your skillset
   - Add or remove skill cards as needed

3. **Add Your Projects**:
   - Replace the sample projects with your actual work
   - Update project titles, descriptions, and tags
   - Link to your live projects or repositories

4. **Styling**:
   - Modify `styles.css` to change colors, fonts, or layout
   - Update CSS variables in `:root` for global color changes

## Deployment

### GitHub Pages

1. Go to your repository settings
2. Navigate to "Pages" section
3. Under "Source", select the branch (usually `main`)
4. Click "Save"
5. Your site will be available at `https://ty202500368-yabs.github.io/Personal-Site/`

### Other Hosting Options

This is a static site and can be deployed to:
- Netlify
- Vercel
- Firebase Hosting
- AWS S3
- Any static hosting service
